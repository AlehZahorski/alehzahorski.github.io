# Personal Portfolio (2019)

Live version:
https://alehzahorski.github.io/

## Overview

This repository contains the source code for my personal portfolio website created in **2019**.

The project was built during a time when AI coding assistants were not widely available.
Every part of the website — structure, styling, and logic — was written manually with a full understanding of how the code works.

The portfolio presents selected projects, technical skills, and contact information, and serves as a central place where recruiters and collaborators can explore my work.

The website is hosted using **GitHub Pages**.

## Features

* Responsive layout
* Project showcase
* About section
* Contact information
* Static deployment with GitHub Pages
